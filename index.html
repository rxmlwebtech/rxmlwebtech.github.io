XML and Web Techologies for Data Science with R

This is the Web site for the book
XML and Web Techologies for Data Science with R
available from Springer as part of the UseR! series.

This site contains code from the book along with additional examples and updates.

The book covers aspects of 30 different R packages we have developed and also several others that we use as examples. The primary "workhorse" packages are

  • XML
  • RCurl
  • RJSONIO
  • Rcompression
  • RHTMLForms
  • SSOAP
  • XMLRPC
  • XMLSchema
  • WADL
  • RXQuery
  • SVGAnnotation
  • RKML
  • RKMLDevice
  • R2GoogleMaps
  • RAmazonS3
  • RAmazonDBREST
  • RGoogleStorage
  • RGoogleDocs
  • RWordXML
  • RExcelXML
  • ROOXML
  • ROpenOffice
  • XDocTools
  • Sxslt
  • XDynDocs
  • RTidyHTML
  • RCSS
  • RElasticSearch
  • RUbigraph
  • RGoogleTrends
