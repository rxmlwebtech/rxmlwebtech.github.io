XML and Web Techologies for Data Science with R

This is the Web site for the book
XML and Web Techologies for Data Science with R
available from Springer as part of the UseR! series.

This site contains code from the book along with additional examples and updates.

The book covers aspects of 30 different R packages we have developed and also several others. The primary "workhorse" packages are

  • XML
  • RCurl
  • RJSONIO
  • XMLSchema
  • RHTMLForms
  • SSOAP
  • XMLRPC
  • SVGAnnotation
  • RKML
  • RKMLDevice
  • RWordXML
  • RExcelXML
  • ROOXML
